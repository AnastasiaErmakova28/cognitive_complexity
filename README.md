# cognitive_complexity
## Разработка методов оценки и снижения когнитивной сложности текстовых материалов
### О проекте
Данный репозиторий содержит программную реализацию научно-исследовательского проекта по методам оценки и алгоритмическому снижению когнитивной сложности русскоязычных текстовых материалов.

Исследование решает фундаментальную проблему когнитивной перегрузки при чтении специализированной документации. В отличие от традиционных методов лексического упрощения (замены сложных слов на простые), которые ведут к потере смысла и фактологическим ошибкам, наш алгоритм фокусируется на синтаксических изменениях.

Мы снижаем среднюю длину синтаксических зависимостей (MDD) и уровень полипредикативности, что позволяет уменьшить когнитивную нагрузку на рабочую память читателя, сохраняя исходный технический смысл.
